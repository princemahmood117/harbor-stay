# 🏨 Harbor Stay

A modern hotel booking web application that allows users to explore rooms, view details, and book stays seamlessly. Built with a focus on smooth UI/UX, responsive design, and efficient booking flow.

🔗 Live Demo: https://harbor-stay-front.vercel.app/
